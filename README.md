# Quick Start

This frontend project is built with `React` and `Vite`. The easiest way to run it is with one command: `npm run start`. This command uses the included `start.sh` script, installs the dependencies if needed, and then starts the development server.

When the server starts, open the local URL shown in the terminal, usually something like `http://127.0.0.1:5173/`. If you want to run the steps manually, use `npm install` first and then `npm run dev`. If you want a production build, run `npm run build`. The main frontend files are in `src/App.jsx` and `src/styles.css`.
